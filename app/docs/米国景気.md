# 米国景気警戒スコアリングシステム

## 概要

100点満点のリセッションリスクスコア。点数が高いほど景気悪化リスクが高い。

- **雇用カテゴリ (50点)**: NFPトレンド(25) + サームルール(15) + 失業保険(2) + 雇用乖離(8)
- **消費カテゴリ (25点)**: 実質個人所得(10) + 消費者信頼感(5) + クレカ延滞率(5) + インフレ乖離(5)
- **構造カテゴリ (25点)**: 求人倍率(10) + U6-U3(7) + 労働参加率(5) + K字型Proxy(3)

### フェーズ分類

| スコア | フェーズ | 上限 | 説明 | アクション |
|--------|----------|------|------|-----------|
| 0-20 | EXPANSION（拡大期） | 80% | 雇用市場は力強く拡大中。過熱リスクに注意。 | 過熱警戒。利確・回転を意識。ポジション上限80% |
| 21-40 | SLOWDOWN（減速期） | 100% | 最良の買い場。バックテスト勝率81%、平均+8.4%/6ヶ月。 | 積極投資OK。フルポジション可 |
| 41-60 | CAUTION（警戒期） | 70% | 複数の指標が悪化傾向。景気後退リスクが高まっている。 | 現物のみ。新規ポジション抑制。ポジション上限70% |
| 61-80 | CONTRACTION（収縮期） | 40% | 景気後退入りの可能性が高い。最も危険なフェーズ。 | 信用取引禁止。現金比率引き上げ。ポジション上限40% |
| 81-100 | CRISIS（危機） | 60% | 深刻な景気後退。ただし底値圏のため逆張りチャンス。 | 分割で現物仕込み。底値圏の逆張り。ポジション上限60% |

### 除外按分（正規化）

未実装・データなしの項目は按分で除外し、100点に正規化：
```
total_score = raw_total / active_max × 100
```
例: K字型(3pt)とインフレ乖離(5pt)が無効 → active_max = 92 → raw 46pt → normalized 50pt

### 4. 雇用乖離 (8点)

**データソース**:
- ADP雇用変化（manual_inputs テーブル、月次）
- Challenger人員削減（manual_inputs テーブル、月次）
- 新規失業保険申請（FRED ICSA）

**計算**:
1. 各指標のギャップを算出（ADP: NFPとの差、Challenger: 適応閾値超過、ICSA: 水準）
2. 重み付け合成: ADP×0.6 + Challenger×0.8(or1.0) + ICSA×0.3
3. Sigmoid変換: `100 / (1 + exp(-weighted_gap / 30))`
4. 8点スケールに変換

**Challenger適応閾値**:
- 直近3ヶ月平均 × 1.3 を閾値とする
- 超過時に trend_flag = True → 重み 0.8 → 1.0
- YoY 3ヶ月平均も補助判定

| Sigmoid出力 | スコア |
|-------------|--------|
| ≥ 70 | 8 |
| ≥ 60 | 5 |
| ≥ 50 | 3 |
| < 50 | 0 |

**現在値**: ADP乖離のみで72/100 → 8pt(MAX)
**注意**: ADP単体でMAXに達することがある。Challengerが効いていない月はADP依存度が高い。

**ダッシュボード表示例**:
| スコア | 表示テキスト |
|--------|-------------|
| 0pt | `ADP等とNFPの乖離度 40%（70%超で警戒、参照: ADP, Challenger, ICSA）` |
| 3pt | `ADP等とNFPの乖離度 55%（70%超で警戒、参照: ADP, ICSA）` |
| 8pt | `ADP等とNFPの乖離度 72%（70%超で警戒、参照: ADP）` |
| データなし | `NFPデータなし` or `代替データ不足` |

**注意**: JOLTSは2ヶ月遅延で発表される遅行指標。発表時点では既に状況が変わっている可能性。

### 12. K字型Proxy (3点)

**データソース**: Yahoo Finance ^RUT (Russell 2000) ÷ ^GSPC (S&P 500)
**計算**: RUT/SPX比率の絶対水準

| 条件 | スコア | 意味 |
|------|--------|------|
| < 0.40 | 3 | 極端なK字型（歴史的P10未満） |
| < 0.45 | 2 | K字型拡大中（P20程度） |
| < 0.50 | 1 | やや乖離あり |
| ≥ 0.50 | 0 | 健全 |

**設計意図**:
- Small Cap(Russell 2000=実体経済) vs Large Cap(S&P 500=MEGA CAP/ハイテク) の乖離
- 比率が低い = 実体経済が金融経済に置いていかれている = K字型経済

**歴史的推移**:

| 2017-2018 | 0.58 | 0pt (健全) |
| 2020 | 0.47 | 1pt (COVID後に乖離拡大) |
| 2022 | 0.46 | 1pt (利上げで小型株打撃) |
| 2023 | 0.43 | 2pt (AI/MEGA CAP集中) |
| 2024 | 0.39 | 3pt (過去最大の乖離) |
| **2025-2026** | **0.36-0.39** | **3pt (歴史的最低水準)** |

**現在値**: 0.386 (パーセンタイル P9.4%) → 3pt(MAX)

**ダッシュボード表示例**:
| スコア | 表示テキスト |

## モデルの強みと限界

### 強み
1. **偽陽性ゼロ**: 2019年（逆イールド）、2022年（ベアマーケット）で正しくEXPANSION維持
2. **COVIDの即座検出**: 1ヶ月でCAUTION、2ヶ月でCRISIS
3. **構造変化の正しい反映**: 2014年のGFC後回復期→2018年ピーク→2024年悪化を正確にトラッキング
4. **K字型の捕捉**: RUT/SPX比率で2023年以降のMEGA CAP集中を検出

### 限界
1. **外生ショックの事前予測は不可能**: COVID(2020-02: 3pt → 2020-03: 54pt)
2. **UMCSENTの政治バイアス**: 政権交代でYoYが大きく振れる。現在5/5だが一部は政治的
3. **JOLTSの2ヶ月遅延**: 発表時点では既に状況が変わっている可能性
4. **手動データ依存**: ADP、Challenger、Truflationは手動入力。未入力時は按分除外されるが感度低下
5. **Sahm初の偽陽性?**: 2024年にSahm=0.53でトリガーしたがリセッション入りせず（まだ確定ではない）
